# Oakridge 🎓

**Oakridge** is a premium, AI-powered attendance management suite designed for modern students. It combines sleek design with intelligent automation to help you stay on top of your academic lifestyle.

![Oakridge Dashboard](https://api.dicebear.com/7.x/shapes/svg?seed=Oakridge)

## ✨ Features

- **AI-Powered Insights**: Get brief, actionable summaries of your academic health.
- **Automated Notifications**: Never miss a low attendance warning or upcoming exam.
- **Intelligent Auto-Fill**: Missed a day of logs? Oakridge can auto-fill them based on your timetable.
- **Engagement Mix**: Visualize your attendance patterns with beautiful charts.
- **Mobile-First Design**: A complete, responsive experience from desktop to mobile.
- **Smart Calender**: Manage holidays and schedule overrides with ease.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite, Framer Motion, Recharts, Tailwind CSS.
- **Backend**: Node.js, Express, MongoDB.
- **State Management**: Zustand.
- **Icons**: Lucide React.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (Running locally or Atlas)

### Setup

1. **Clone the Repo**
   ```bash
   git clone https://github.com/Sagararora90/oakridge.git
   cd oakridge
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file with your MONGO_URI
   node server.js
   ```

3. **Frontend Setup**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
